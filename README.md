# 数据结构基础

本书是一份面向零基础读者的数据结构学习文档。内容覆盖从最基本的数组、链表，到树、图、哈希表等核心数据结构，以及常用的排序与查找算法。

## 目标读者

- 没有数据结构基础，但具备基本编程能力的学习者
- 正在准备技术面试、需要系统复习数据结构的开发者
- 希望补齐计算机科学基础知识的自学者

## 学习建议

1. 按照章节顺序阅读，每一部分都建立在前一部分的基础之上
2. 动手实现每个数据结构，不要只停留在"看懂了"的层面
3. 完成每章末尾的思考题，检验自己的理解程度
4. 遇到不理解的概念，回到前置章节复习

## 内容结构

本书分为四个部分：

- **前言** — 数据结构的基本概念与算法分析基础
- **第一部分：线性结构** — 数组、链表、栈、队列、字符串
- **第二部分：树形结构** — 二叉树、二叉搜索树、平衡树、堆
- **第三部分：散列与图** — 哈希表、图的存储与遍历
- **第四部分：排序与查找** — 常用排序算法与查找算法

## 代码说明

本书中的代码示例以伪代码为主，辅以 Python 实现，便于零基础读者理解。
